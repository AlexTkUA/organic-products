Структура проекту


1. В компонентів header.js лежить функція яка відмальовує header на кожній сторінці. 

2. В компонентів footer.js лежить функція яка рендерить footer на кожній сторінці. На кожній сторінці має бути <main class = "main"></main> для того що footer був вставлений в правильнемо місці. 

3. В файлі url.js лежить функція яка повертає правильний шлях до файлів. Функція працює як для life server так і для gitHub але це рішення є костилемо.

4. В компонентів renderBanner лежить функція яка рендерить банер на сторінках url яких мають рядок "/pages/". В цьому компоненті лежить об'єкт з ключами до інших вкладнених об'єктів. Назва ключа МАЄ БУТИ ТАКОЮ Ж ЯК НАЗВА ПАПКИ ПІСЛЯ "/pages/".
